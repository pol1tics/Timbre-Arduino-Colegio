Proyecto Timbre Inteligente 5.0

Automatización para la Puntualidad y Seguridad Escolar.

1. La Misión: ¿Por qué creamos esto?

El objetivo es simple: reemplazar el viejo timbre manual por un sistema **100% automático, preciso y seguro**.

El Problema (Antes)

  • Depende de errores humanos (olvidos).
  • Retrasos de minutos en los horarios.
  • Sin sistema de pánico centralizado.
  • Poca flexibilidad.

La Solución (Ahora)

  • 100% Automático y confiable.
  • Precisión al segundo (con el RTC).
  • Botón de Alarma de Emergencia.
  • Fácil de programar y mantener.

2. Los Héroes del Hardware (¡Desliza!)

Estos son los componentes que hacen la magia. ¡En un celular, puedes deslizar las tarjetas!

Arduino UNO

El Cerebro. Lee nuestro código y da las órdenes de "Tocar ahora" o "Esperar". Usa el Pin Digital 7 para la orden al timbre.

Módulo RTC

El Reloj Atómico. Mantiene la hora exacta siempre, sin fallar, incluso cuando la luz se va.

Módulo Relé

El Interruptor de Poder. Recibe la orden del Arduino y conecta la electricidad grande para hacer sonar la campana real.

LCD y Botón

El Monitor y el Pánico. La pantalla LCD nos muestra la hora. El botón de pánico es la entrada que le dice al Arduino: "¡ALERTA AHORA!".

Diagrama (Fritzing)

El mapa que nos dice cómo conectar todo.

4. El Código: ¡Traductor de Programador!

El código es la **receta** que sigue el Arduino. ¡No te asustes! Es más fácil de lo que parece. Aquí tienes la "traducción" de las órdenes más importantes.

Código (Lo que escribimos) Traducción (Lo que el Arduino entiende)
void loop() { ... } "Vas a repetir estas órdenes para siempre, sin parar. Es tu único trabajo."
DateTime now = RTC.now(); "Oye Reloj RTC, ¡dame la hora exacta AHORA MISMO!"
if (hay_una_emergencia) { ... } "Revisa si el botón de pánico está presionado. Si lo está, ¡ignora todo lo demás y ejecuta el plan de ALARMA!"
activar_timbre(DURACION_LARGA); "¡Toca la sirena AHORA! Y hazlo por 5 segundos."
else { ... } "Si NADIE presionó el botón de pánico, entonces sigue con el plan normal..."
verificar_horario(horario1); "Revisa la lista de horarios de clase. Si la hora del reloj coincide con la lista, ¡toca el timbre normal!"
digitalWrite(PIN_TIMBRE, HIGH); "¡Envía energía al Pin 7! (Esto enciende el Relé y la campana suena)"
digitalWrite(PIN_TIMBRE, LOW); "¡Corta la energía del Pin 7! (Esto apaga el Relé y la campana se calla)"

5. Preguntas Frecuentes (FAQ)

Aquí respondemos las dudas más comunes. ¡Haz clic en una pregunta para ver la respuesta!

¿Qué pasa si se va la luz en el colegio?

¡Nada! El Módulo RTC (el reloj) tiene su propia batería de moneda (como la de un reloj de pulsera). Cuando la luz regrese, el Arduino se encenderá y le preguntará la hora correcta al RTC. El sistema seguirá funcionando sin problemas.

¿Fue muy caro hacer este proyecto?

No. Lo genial del Arduino es que es tecnología accesible. Todos los componentes (Arduino, Relé, RTC, LCD y cables) son económicos y fáciles de conseguir en tiendas de electrónica en Colombia o por internet.

¿Es muy difícil programar el Arduino?

¡Para nada! Si te gusta la tecnología y los videojuegos, es muy similar. Usamos un lenguaje basado en C++ que es muy lógico. Como viste en la tabla, es solo dar órdenes como "Encender" o "Esperar".

6. Visión a Futuro y los Creadores

Este proyecto es solo el comienzo. La automatización puede ir más allá.

Futuro: Control Wi-Fi

La próxima versión podría incluir un módulo Wi-Fi (ESP8266) para cambiar los horarios desde una página web o un celular, sin necesidad de conectar el computador.

Futuro: Alertas SMS

Se podría añadir un módulo GSM para que, al presionar el botón de pánico, el sistema envíe un mensaje de texto de alerta a los directivos o a los servicios de emergencia.
